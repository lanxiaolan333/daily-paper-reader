## 摘要
现有的基于视觉基础模型（VFM）的迭代立体匹配流水线未能充分利用三条信息通路：多尺度骨干特征被折叠为单层级相关性、几何先验在初始化时未被利用、上下文仅局部传播。这些缺陷在光度线索退化时加剧，使得水下场景成为一个严格的一般化测试。为此，我们提出LinStereo，它基于Depth Anything V3构建，其核心是位置感知线性注意力（PALA）模块，该模块以线性代价用全局聚合替代局部循环，将可靠估计从良好匹配区域传播到退化区域，同时保持视差结构。PALA的有效性由两个关键组件保障：层级语义代价体（HSCV），它从VFM特征层级中提供尺度对齐的相关性；以及深度先验初始化（DPI），它将单目深度转换为度量校准的暖启动。LinStereo在标准基准上达到最先进水平的精度，并展现出强大的跨域泛化能力，尤其是在水下场景中——严重的光度退化使得立体匹配极具挑战性——在TartanAir-UW数据集上AbsRel降低28%，在真实世界水下数据集SQUID上降低26%，取得了最佳整体精度和一致的性能提升。

## Abstract
Existing Vision Foundation Model (VFM)-based iterative stereo pipelines under-exploit three information pathways: multi-scale backbone features are collapsed into single-level correlations, geometric priors remain untapped at initialization, and context propagates only locally. These gaps widen under degraded photometric cues, making underwater scenes a stringent generalization test. To address this, we propose LinStereo, built upon Depth Anything V3, whose core is a Position-Aware Linear Attention (PALA) module that replaces local recurrence with global aggregation at linear cost, propagating reliable estimates from well-matched regions into degraded areas while preserving disparity structure. PALA is made effective by two enabling components: Hierarchical Semantic Cost Volumes (HSCV), which supply scale-aligned correlations from the VFM feature hierarchy, and a Depth Prior Initialization (DPI) that converts monocular depth into a metrically calibrated warm start. LinStereo achieves state-of-the-art-level accuracy on standard benchmarks and strong cross-domain generalization, particularly on underwater scene where severe photometric degradation makes stereo matching particularly challenging, attaining the best overall accuracy with consistent gains 28% lower AbsRel on TartanAir-UW, 26% on SQUID, a real-world underwater dataset).

### 2. 论文提出的方法论：核心思想、关键技术细节
- **核心思想**：基于**位置感知线性注意力（PALA）** 替换传统局部循环更新算子，以线性复杂度实现全局信息传播；配合**层次语义代价体（HSCV）** 和**深度先验初始化（DPI）**，使全局聚合有效。
- **整体框架**：采用冻结的 Depth Anything V3（DA3）作为共享骨干，输出多尺度特征（s=4,8,16）及单目深度估计；解码器由三个 PALA 更新器（每尺度一个）组成，在迭代中逐步细化视差。
- **关键技术细节**：
  - **HSCV**：对每个尺度的特征构建 1D 相关性体，并在视差维度构建 4 级金字塔。更新器在每次迭代中可重新查询任意尺度，而非一次性融合。
  - **DPI**：利用 SIFT 匹配获取稀疏对应点，通过最小二乘估计单目深度的尺度和偏移（d = α/D̂_mono + β），转化为度量校准的初始视差；若内点数 <20 则回退零初始化。
  - **PALA 更新器**：采用线性注意力（Q-K 乘积 → K⊤V → Q(K⊤V)）达到 O(N) 复杂度。引入非对称 2D RoPE 恢复空间位置信息（仅在分子使用位置编码，分母保持原始归一化），并加入局部卷积和门控更新机制。
  - **损失函数**：包含 L1 损失、边缘感知平滑损失和多尺度梯度损失，各迭代步按指数加权（γ=0.9）。
- **算法流程**（文字说明）：输入立体图像对 → DA3 提取多尺度特征和单目深度 → DPI 生成初始视差 → 每迭代步：对各尺度，从 HSCV 中查找当前视差附近的相关性，编码为运动特征；三个 PALA 更新器按粗→细顺序更新隐藏状态 → 最后一层隐藏状态解码为视差残差。总迭代 T=8。

- LinStereo 在仅用 SceneFlow 训练的条件下，在标准基准上达到与使用额外数据的方法竞争乃至超越的精度（尤其在遮挡区域 EPE 降低 37%）。

### 7. 优点
- **方法创新**：首次将线性注意力引入迭代立体匹配，并通过非对称 RoPE 解决位置信息丢失问题，设计精巧。
- **效率与性能平衡**：通过减少迭代次数实现加速，而非牺牲模型容量，在低迭代（T=2）时仍优于多数高效方法。

- **消融充分**：从设计选择到超参数均进行系统分析，可信度高。
- **开源贡献**：代码、SeaStereo-Dataset 和实验室水箱数据集均公开，促进可复现研究。

### 8. 不足与局限
- **骨架依赖性**：DA3 骨干冻结且占 94% 参数量，限制了在资源受限设备上的部署（T=8 时 80ms/帧，较轻量级方法慢）。
