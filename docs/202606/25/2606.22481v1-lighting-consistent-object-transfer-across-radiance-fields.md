## 摘要
3D高斯泼溅（3DGS）被广泛用于捕捉和渲染真实场景。将一个捕捉中的对象合成到另一个捕捉中在视觉特效、建筑与室内设计或市场营销等多个领域都有应用。然而，由于两个场景之间的光照条件不同，从源场景中提取对象并简单粘贴到目标场景中会产生不真实的结果。为了解决这个问题，我们引入了一个扩散模型，该模型可以协调光照不一致的简单合成图像。该模型使用图像对（不一致的合成输入、一致的输出）的异构数据集进行训练，结合了合成、生成和真实数据。我们的完整3D解决方案允许用户从源场景中提取对象并将其合成到目标场景中。由此，渲染出带有合成对象的目标场景的（不一致）视图。我们的扩散模型协调这些视图中的每一个，最后通过后优化步骤将其整合到3DGS表示中。我们的方法提供了视觉上引人注目的结果，使得3DGS之间的对象迁移易于使用，并且与之前的方法相比显著提高了质量。

## Abstract
3D Gaussian Splatting (3DGS) is widely used to capture and render real scenes. Compositing objects from one capture into another has applications in many domains, such as VFX, architecture and interior design, or marketing. However, extracting an object from a source scene and naively pasting it into a target scene will fail to produce realistic results due to the different lighting conditions between the two scenes. To address this problem, we introduce a diffusion model that harmonizes naively composited images with inconsistent lighting. The model is trained with a heterogeneous dataset of image pairs (inconsistent composite input, consistent output), combining synthetic, generated, and real data. Our complete 3D solution allows a user to extract an object from the source scene and composite it into the target scene. From this, the (inconsistent) views of the target scene with the composite object are rendered. Our diffusion model harmonizes each one of these views, which are finally consolidated in a 3DGS representation with a post-optimization step. Our method provides visually compelling results, making object transfer between 3DGS easy to use and significantly improving quality compared to previous methods.

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：在3D高斯泼溅（3DGS）场景中，将物体从一个场景迁移到另一个场景时，由于源场景和目标场景的光照条件不同，简单复制粘贴会导致合成图像光照不一致（缺失阴影、反射不匹配），视觉效果不真实。
- **整体含义**：本文提出DOT3D（Diffusion Object Transfer in 3D）方法，利用扩散模型对单视图进行光照协调，再通过3DGS后优化得到多视图一致的输出，从而实现场景间物体的光照一致迁移。该方法简化了传统需专家建模和光照估计的流程，为视觉特效、室内设计、营销等领域提供易用的工具。

## 2. 方法论
- **核心思想**：两阶段流水线：（1）单视图协调：用微调的扩散模型将简单合成的不一致图像转化为光照协调的图像；（2）多视图整合：对独立协调的视图进行3DGS后优化，强制多视图一致性。
- **关键技术细节**：
  - **交互式对象提取与放置**：利用BiRefNet获取源物体2D掩码，通过为高斯原语添加二进制特征进行3D分割，用户通过界面调整物体位置、方向和缩放，生成初始不一致合成场景。
  - **单视图协调扩散模型**：
    - 基于FLUX.1-schnell（一种预训练文本到图像模型）进行微调，使用修复流匹配损失函数：  
      \[ L_\theta = \mathbb{E} \left[ \| v_\theta(z_t, z_{\text{comp}}, z_{\text{mask}}, c, t) - (\epsilon - z_0) \|_2^2 \right] \]
      其中 \(z_t\) 是目标潜变量的加噪版本，\(z_{\text{comp}}\) 和 \(z_{\text{mask}}\) 分别为输入合成图像和物体掩码的潜变量，\(c\) 为空文本条件。网络预测速度 \(v\)，输入采用通道拼接。
    - 训练数据由三部分组成：
      - **合成数据**：30个Blender室内场景，渲染1.9K目标图像，通过不同环境贴图生成51K不一致合成对。
      - **生成数据**：用FLUX生成1.6K目标图像，配合自训练物体移除网络获得背景，用DiffusionRenderer重光源物体生成6.5K不一致对。
      - **真实数据**：从ORIDa数据集子采样2K图像，同样用DiffusionRenderer重光生成8K不一致对。
  - **多视图整合**：
    - 初始化3DGS为简单合成场景，固定几何、不透明度等属性，仅优化颜色（所有球谐波系数，高阶置零）。
    - 使用感知损失（LPIPS）代替像素级光度损失，训练10K迭代，不进行高斯致密化和不透明度重置。
    - 无需迭代数据集更新，单次协调加单次后优化即可。

## 6. 主要结论与发现
- DOT3D能够有效实现光照一致的物体迁移，生成符合目标场景光照、阴影和反射的逼真结果。
- 在定量指标（PSNR、SSIM、LPIPS、FID、KID）上显著优于扩展的逆渲染方法（Gaussian Shader、GS-IR、3DGS-DR）。
- 使用精心构建的异质数据集（合成+生成+真实）训练的扩散模型，比仅使用单一数据源的模型表现更好。
- 单次后优化足以实现多视图一致性，无需复杂的迭代数据集更新策略，节省计算时间。
- 感知损失比光度损失在后优化中保留更多细节。

## 8. 不足与局限
- **实验覆盖**：
  - 定量评估仅限于9个合成场景，真实场景缺乏数值指标（仅定性展示），通用性说服力有限。
  - 2D协调对比中未使用量化指标（如FID）在统一测试集上评估。
  - 对比方法（MV-CoLight等）仅为2D协调，未在3D场景中公平比较3D迁移效果（因这些方法无3D输出）。
- **局限性**：
  - 扩散模型的变分自编码器（VAE）对高频细节物体引入模糊或块状伪影。
  - 协调过程可能过度改变物体外观（如金属变玻璃），影响材质真实性。
  - 无法保证绝对准确的光照协调，仅能生成“合理”结果。
  - 依赖高质量合成的3D场景数据，扩展至更复杂或缺失真实数据的场景仍有难度。
- **其他**：
  - 后优化仅优化颜色，无法改变物体几何或阴影位置，对严重不一致场景可能效果有限。
  - 计算开销：协调模型推理需4秒/图，对大场景（数百视图）总耗时较长。
