## 摘要
从单张单目图像同时进行三维重建和6D物体姿态估计是一个天生不适定问题。然而，在工业场景中，同一物体的多个实例通常随机堆放在料箱中，隐式地在单张图像内提供了该物体的多个视图。我们表明，可以利用这种隐式多视图几何结构来同时重建物体的三维形态并估计每个可见物体实例的6D姿态。我们提出MooMIns，一种基于高斯泼溅的新方法，它反转了原始高斯泼溅公式：不是从多个相机渲染单个场景，而是从单个相机渲染多个物体实例。我们的方法使用SAM3实例分割掩码和改进的运动恢复结构（SfM）流水线进行初始化。与学习的单目深度估计不同，我们基于图像证据进行真正的几何重建，避免了由训练数据先验引起的幻觉。我们在合成和真实料箱抓取场景上评估了MooMIns，并展示了对未见物体的精确重建以及单个实例的可靠姿态估计。

## Abstract
Simultaneous 3D reconstruction and 6D object pose estimation from a single monocular image is an inherently ill-posed problem. In industrial settings, however, multiple instances of an object are often randomly arranged in bins, implicitly providing several views of the same object within a single image. We show that this implicit multi-view geometry can be exploited to simultaneously reconstruct the object in 3D and estimate the 6D pose of each visible object instance. We present MooMIns, a new Gaussian-splatting-based approach that inverts the original Gaussian splatting formulation: instead of rendering a single scene from multiple cameras, we render multiple object instances from a single camera. Our method is initialized with SAM3 instance segmentation masks and a modified Structure from Motion (SfM) pipeline. In contrast to learned monocular depth estimation, we perform true geometry-based reconstruction from image evidence, avoiding hallucinations caused by training data priors. # 2. 方法论

- **核心思想**：反转经典高斯泼溅（3D-GS）框架——不是从多个相机渲染一个场景，而是从一个固定相机渲染多个物体实例，每个实例共享同一个规范化的物体模型（canonical object）及其材质参数。
- **关键技术细节**：
  - **基于反射的球面Voronoi（SV）参数化**：使用2D高斯原语，分离几何与光照（远场环境贴图`U_far`和近场光照探针`U_near`），从而处理反射与纹理物体。
  - **初始化**：
    1. 用SAM3自动分割每个实例的可见掩膜（需用多模态LLM生成粗略物体描述作为提示）。
    2. 对每个掩膜剪切出单实例图像，输入COLMAP（调整参数）获得稀疏点云和虚拟相机位姿，作为规范化物体的初值。
  - **场景生成**：将规范化的高斯原语`G_canon`通过优化得到的候选物体位姿`CP_O`变换并组合成`G_multi`，再从固定单相机`O_CC`渲染。
  - **训练优化**：仅优化`G_canon`、物体位姿`CP_O`、光照`U`，而相机位姿固定；损失函数含光度损失`L_p`（L1+SSIM）、二元交叉熵掩膜损失`L_m`、2D-GS正则化项`L_n`和`L_d`。
  - **适应性密度控制（ADC）修正**：由于每轮迭代每个高斯可被最多J个实例观测，将阈值条件改为跨所有实例求和。
  - **实例Dropout**：每次迭代随机丢弃60%实例，防止多实例混叠，并利用随机背景色避免黑色高斯“隐藏”。
  - **鲁棒实例移除**：基于实例中位损失与稳健标准差σ_r，移除高误差实例（限制最多20%）。
  - **稀疏视图处理**：对规范化的高斯原语施加30%常数丢弃率，防止过拟合。
- **算法流程**：输入单张多实例RGB图像+相机内参 → SAM3分割 → COLMAP初始化 → 修改后的GS管线迭代优化（ADAM更新位姿、高斯原语、光照） → 输出规范化物体3D模型+每个实例6D位姿。

# 6. 主要结论与发现

- MooMIns能在零样本（zero-shot）条件下，从单张多实例RGB图像中联合重建规范物体3D模型并估计每个实例的6D姿态。
- 对于有纹理、无遮挡的物体，初始化（SAM3 + COLMAP）可以直接成功，获得良好结果（SYN1：MRE 0.98°, MTE 14.17 mm, d_c 1.31 mm）。
- 对于纹理less、高反射的工业件（XYZ1-5），COLMAP初始化失败，但若提供近似球体初值和带噪声的GT位姿（5°旋转噪声+30 mm平移噪声），改进后的GS管线仍能显著优化位姿和几何（如XYZ1：MTE从30.00降至1.66，d_c从32.55降至0.78）。
- 消融表明：掩膜损失`L_m`、实例dropout、远/近场光照建模对工业场景至关重要。
- 鲁棒性分析显示，改进的GS管线对初始位姿噪声容忍度较高（旋转≤15°, 平移≤150 mm）。

# 7. 优点

- **任务创新**：首次定义并解决“多实例单图3D重建+物体姿态估计”任务，利用隐式多视图几何突破单目病态限制。
- **方法设计合理**：通过反转渲染方向（单相机多实例）避免了光照不一致问题，使整体场景光照一致。
- **零样本能力**：无需物体特定预训练或CAD模型，仅依赖SAM3+COLMAP初始化，对新物体具有泛化性。
- **可处理反射与纹理less物体**：通过基于反射的SV参数化分离几何与光照，可处理工业中常见的反射无纹理物体。
- **鲁棒机制丰富**：实例dropout、鲁棒实例移除、稀疏视图Dropout等策略有效防止过拟合和实例混淆。

# 8. 不足与局限

- **依赖初始化质量**：纹理less金属件上COLMAP失败，需要人为提供近似球体初值和噪声位姿，尚未实现真正全自动初始化。
- **需要额外约束**：目前需已知相机内参，与任务纯形式（仅输入单张RGB）存在差距；未来计划消除此约束。
- **无法自动生成amodal掩膜**：处理遮挡需要amodal掩膜，当前仅能获得可见掩膜（SAM3），实际中需依赖振动给料器避免遮挡或后续扩展优化生成amodal掩膜。

  - 仅使用单张图像的评估（每次实验只处理一张多实例图），未验证不同实例数量、不同背景、不同光照条件下的鲁棒性。
